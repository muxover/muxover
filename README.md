## What I Build

Most of it is private. Here's the territory:

- **Network & Infrastructure** — proxies, traffic interception, TCP tuning, egress binding, HTTP pipelining
- **Security Research** — SSL unpinning, binary analysis, behavioral intelligence extraction
- **Automation & Bots** — Telegram-controlled pipelines, high-throughput CLI tools, scrapers
- **SaaS & Products** — B2B systems, payment integrations, full-stack platforms under Jax Development LLC
- **AI Pipelines** — automated content systems, Whisper transcription, LLM-powered workflows
